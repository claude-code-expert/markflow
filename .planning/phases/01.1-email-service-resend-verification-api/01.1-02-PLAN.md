   ```typescript
   async function resendVerification(email: string) {
     const found = await db
       .select()
       .from(users)
       .where(eq(users.email, email.toLowerCase()))
       .limit(1);

     const user = found[0];

     // 사용자 없거나 이미 인증됨 → 동일 응답 (이메일 열거 방지, forgotPassword()와 동일 패턴)
     if (!user || user.emailVerified) {
       return { sent: true };
     }

     // 새 토큰 발급 (기존 토큰 재사용 X — 보안상 이전 링크 무효화)
     const emailVerifyToken = crypto.randomUUID();
     const emailVerifyExpiresAt = new Date(Date.now() + 24 * 60 * 60 * 1000);

     await db
       .update(users)
       .set({
         emailVerifyToken,
         emailVerifyExpiresAt,
         updatedAt: new Date(),
       })
       .where(eq(users.id, user.id));

     const verifyUrl = `${FRONTEND_URL}/verify-email?token=${emailVerifyToken}`;
     await sendEmail({
       to: user.email,
       subject: 'MarkFlow 이메일 인증',
       html: verificationEmailHtml(verifyUrl),
     });

     return { sent: true };
   }
   ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → resend-verification API | 인증 없는 공개 엔드포인트, rate limit으로 보호 |
| URL searchParam → API body | 프론트엔드에서 email을 URL param에서 읽어 API에 전달 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01.1-05 | Information Disclosure | resendVerification() | mitigate | 사용자 존재 여부를 응답으로 구분 불가. 항상 `{ sent: true }` 반환 (forgotPassword()와 동일 패턴) |
| T-01.1-06 | Denial of Service | POST /resend-verification | mitigate | authRateLimit() 적용: IP 기준 10회/15분. Resend 무료 티어 3,000통/월 한도로 추가 보호 |
| T-01.1-07 | Spoofing | emailVerifyToken | mitigate | crypto.randomUUID() (UUID v4, 122-bit entropy). 24시간 만료. 재발송 시 이전 토큰 무효화 (새 토큰으로 교체) |
| T-01.1-08 | Information Disclosure | URL에 email 노출 | accept | `/verify-email?email=xxx` — 사용자 본인의 이메일이 본인의 브라우저 URL에 노출. 제3자 공유 위험은 낮음. 브라우저 히스토리에 남지만 로그인 페이지도 동일. 대안(localStorage)보다 단순하고 탭 간 이동 안정적 |
</threat_model>
