🎃 AI Coding Terminal

Kaku
Fast, Out-of-the-Box Terminal for AI Coding

WezTerm 기반의 AI 코딩 특화 macOS 터미널. 제로 설정으로 즉시 사용 가능하며, Lua 기반 완전 커스터마이징을 지원합니다.

🦀 Rust
🍎 macOS 전용
⚡ ~40MB 바이너리
🌙 Shell 부트스트랩 ~100ms

설정 파일 (kaku.lua)
📄 기본 설정 예시 — ~/.config/kaku/kaku.lua
-- Kaku 사용자 설정 (WezTerm Lua API 완전 호환)
-- 이 파일이 없으면 앱 내부 기본값이 자동 적용됨

local config = {}

-- 테마 / 색상
config.color_scheme = "Catppuccin Mocha"

-- 폰트 설정
config.font = wezterm.font("JetBrains Mono")
config.font_size = 14.0

-- 투명도 (0.0 ~ 1.0)
config.window_background_opacity = 0.92
config.macos_window_background_blur = 20

-- 선택 시 자동 클립보드 복사 비활성화
config.copy_on_select = false

-- GPU 가속 (VM 환경 문제 시)
config.front_end = "WebGpu"

return config
⚙️ 주요 설정 옵션
-- 탭바 스타일
config.tab_bar_at_bottom = false
config.use_fancy_tab_bar = true

-- 스크롤바 (kaku config TUI에서도 설정 가능)
config.enable_scroll_bar = true

-- 패인 사이 간격
config.window_padding = {
  left = 8, right = 8,
  top = 8, bottom = 0,
}

-- 시작 창 크기
config.initial_cols = 220
config.initial_rows = 50

-- 맥 OS 폰트 렌더링 최적화
config.freetype_load_flags = "NO_HINTING"
config.freetype_render_target = "Light"

return config
⚡ 설정 TUI (권장 방법)
파일을 직접 편집하는 대신 kaku config 또는 Cmd + , 로 설정 TUI를 열 수 있습니다. 폰트, 테마, 투명도, 알림 벨, 스크롤바, Kaku Assistant 등 주요 옵션을 GUI로 편집하며, 아래 화살표 키로 옵션을 이동하고 Enter로 선택합니다.

자주 묻는 질문
Homebrew 패키지명이 왜 kakuku인가요?
Homebrew 공식 저장소에 kaku라는 이름의 방치된 음악 플레이어 패키지가 이미 존재합니다. 충돌을 피해 kakuku로 명명했으며, 기억하기 쉬운 귀여운 변형입니다.
Windows나 Linux 버전이 있나요?
현재는 macOS 전용입니다. macOS 버전이 성숙해지면 이후 Windows/Linux 지원을 검토할 예정입니다.
투명 배경 창을 사용하고 싶습니다.
~/.config/kaku/kaku.lua에 config.window_background_opacity = 0.85와 config.macos_window_background_blur = 20을 추가합니다. 상단/오른쪽/하단 패딩 영역도 시각적으로 일관되게 처리됩니다.
가상 macOS 환경에서 실행이 안 됩니다.
VM에 사용 가능한 GPU 백엔드가 없을 때 발생합니다 (failed to create NSOpenGLPixelFormat). VM GPU 가속을 활성화하거나, ~/.config/kaku/kaku.lua에 config.front_end = 'WebGpu'를 추가하세요.
선택 시 자동 복사 기능을 끄고 싶습니다.
~/.config/kaku/kaku.lua에 config.copy_on_select = false를 추가하면 자동 클립보드 복사와 확인 토스트가 모두 비활성화됩니다.
WezTerm 설정 파일을 그대로 사용할 수 있나요?
네. Kaku는 WezTerm의 Lua API와 100% 호환됩니다. 기존 wezterm.lua 설정을 ~/.config/kaku/kaku.lua에 복사하면 별도 마이그레이션 없이 사용 가능합니다.
