📌 3가지 연결 시나리오
🆓시나리오 A · 무료 도메인
DigitalPlat에서 무료 .dpdns.org 도메인을 발급받고 Cloudflare에 연결합니다. 도메인이 없어도 즉시 시작 가능.
🔄시나리오 B · 기존 도메인 이전
이미 보유한 도메인의 네임서버를 Cloudflare로 변경합니다. 풀 DNS 관리 & 자동 SSL.
📌시나리오 C · CNAME만 추가
현재 DNS 레지스트라에 CNAME 레코드만 추가합니다. 기존 DNS 설정 유지하며 최소 변경.
🔒 보안 상태
Cloudflare Proxy✓ 활성화 (DDoS 차단)
SSL 모드✓ Full (Strict)
포트 노출✓ 없음 (Tunnel 전용)
HSTS✓ 활성화
