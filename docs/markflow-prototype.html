# Getting Started with MarkFlow ## Overview MarkFlow is a **markdown-first** knowledge management platform for teams. ## Key Features - [[Dual Editor]] with live preview - Real-time collaboration - CSS theme per workspace - Embed anywhere with NPM/iframe
Getting Started with MarkFlow
Overview

MarkFlow is a markdown-first knowledge management platform for teams.

Key Features
  • Dual Editor with live preview
  • Real-time collaboration
  • CSS theme per workspace
  • Embed anywhere
Core Features
마크다운을 팀의 언어로
✏️
Dual-Pane Editor
CodeMirror 6 기반의 소스 편집과 실시간 렌더링 프리뷰. CommonMark + GFM + KaTeX 지원. 스크롤 동기화.
🔗
Document Linking
문서 간 연관 관계, Prev/Next 네비게이션, 위키링크([[문서명]]) 구문으로 지식 그래프 구성.
👥
Real-time Collab
Yjs CRDT 기반 최대 30인 동시 편집. 원격 커서, 인라인 댓글, 스레드형 토론 지원.
🎨
CSS Theme System
워크스페이스 단위 CSS 커스터마이징. 5가지 프리셋 + CodeMirror 기반 직접 편집. CSS 변수 오버라이드.
📦
Embed Anywhere
NPM 패키지(@markflow/editor) · iframe + Guest Token · REST API 세 가지 방식으로 어떤 프로젝트에도 통합.
📥
Import / Export
.md · .zip · HTML · PDF 완전 지원. 폴더 구조 보존 ZIP 내보내기. Puppeteer 기반 서버사이드 PDF.
