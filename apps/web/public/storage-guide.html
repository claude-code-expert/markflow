이미지 업로드 설정 가이드

MarkFlow 에디터에서 이미지 업로드 기능을 사용하려면 Cloudflare R2 저장소와 Worker를 연결해야 합니다. 아래 6단계를 차례대로 따라 하시면 됩니다.

1

Cloudflare 가입

무료 계정을 생성합니다.

cloudflare.com
2

Wrangler 로그인

npx wrangler login
3

R2 버킷 생성

npx wrangler r2 bucket create markflow-images
bucket not found 에러가 뜨면 이 단계부터 먼저 실행하세요.
4

퍼블릭 액세스 + PUBLIC_URL 설정

  1. Cloudflare 대시보드 로그인
  2. 좌측 메뉴 Storage & Databases → R2 Object Storage
  3. 생성한 markflow-images 버킷 선택 → Settings 탭
  4. Public Development URL → Allow Access 활성화
  5. 표시된 URL을 apps/worker/wrangler.toml의 PUBLIC_URL에 입력
wrangler.toml은 프로젝트 루트의 apps/worker/ 폴더에 있습니다.
예시: PUBLIC_URL = "https://pub-YOUR_BUCKET_ID.r2.dev"
5

Worker 배포

cd apps/worker
npx wrangler deploy

배포 후 출력되는 URL을 기억해 두세요.
예시: https://markflow-r2-uploader.account-id.workers.dev

6

Worker URL을 MarkFlow에 입력

5단계에서 받은 URL을 MarkFlow 앱의 설정 → 이미지 저장소 페이지에서 입력하고 연결 테스트 → 저장 순으로 저장합니다.

이후 에디터에서 이미지를 드래그·붙여넣기 하면 자동 업로드됩니다.
